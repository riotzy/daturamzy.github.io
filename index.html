PROGRAMMING SKILLS

HTML

HyperText Markup Language, is the standard markup language used to create and design the structure of web pages. It provides a way to describe the content of a webpage using a system of elements and tags. HTML is not a programming language; instead, it is a markup language that browsers interpret to display content on the web.

CSS

Cascading Style Sheets, is a style sheet language used for describing the presentation of a document written in HTML or XML. CSS defines how elements should be displayed on the screen, in print, or in other media. It allows web developers to control the layout, colors, fonts, spacing, and other visual aspects of web pages.

JAVASCRIPT

JavaScript is a high-level, interpreted programming language that is widely used to make web pages interactive. It allows developers to add dynamic behavior to websites, create interactive interfaces, manipulate the Document Object Model (DOM), and communicate with servers. JavaScript is an essential component of web development and is supported by all major web browsers.

FLASK

Flask is a micro web framework written in Python. It is designed to be lightweight, easy to use, and modular, allowing developers to build web applications quickly and efficiently. Flask is often referred to as a "micro" framework because it does not come with a lot of built-in tools and libraries, but it gives developers the flexibility to choose the components they need for their projects.

REACT

React is a JavaScript library for building user interfaces, particularly for single-page applications where user interactions are frequent and complex. Developed and maintained by Facebook, React allows developers to create reusable UI components that update efficiently and automatically in response to changes in the application's state.
